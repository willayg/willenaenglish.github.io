## Game Modes (Level 2)

### Mode 1: Find the Mistake
**Purpose**: Students identify and correct grammatical errors in sentences
**Implementation**: 
- Display sentences with intentional errors
- Students select/highlight the mistake
- Students correct it
- Provides immediate feedback

**Applicable Topics**: All grammar topics

---

### Mode 2: Translation Mode
**Purpose**: Students translate between English and Korean (or vice versa)
**Implementation**:
- Display sentence in English, translate to Korean
- Display sentence in Korean, translate to English
- Students type or select from options
- Tests comprehension and vocabulary

**Applicable Topics**: All grammar topics
